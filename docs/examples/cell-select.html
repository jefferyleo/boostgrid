Cell selection & copy

Set cellSelection: true and the body becomes spreadsheet-friendly. Click a cell to anchor; shift-click or drag to extend the rectangle; Ctrl/Cmd + C copies the selection as tab-separated values (one row per \n). Esc clears.

Try selecting a few cells below, then press Ctrl+C and paste into your spreadsheet of choice.
Quarter Region Revenue YoY % Account exec 
import { attach } from "boostgrid";

attach("#grid", {
  cellSelection: true, // off by default
});

// Conflicts with rowSelect (the whole-row click handler) —
// pick one or the other.
// Copy output is tab-separated values, newline-separated rows.
// Pasted into Excel / Google Sheets / Numbers, the columns line
// up with the source rectangle.

// Selecting (Q1 NA, Q1 EMEA, Q1 APAC) over the Revenue column
// produces:

Q1\tNA\t$1.24M
Q1\tEMEA\t$0.96M
Q1\tAPAC\t$0.71M

// Tabs and newlines inside cell values are flattened to single
