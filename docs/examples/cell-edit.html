Cell-level edit

Mark a column with data-editable="true". Double-click any cell in that column to swap an editor in. Press Enter or click out to commit, Esc to cancel. The onCellEdit callback fires after every commit and exposes a revert() escape hatch for async server rejections.

<!-- Mark editable columns -->
<th data-column-id="sender"  data-editable="true">Sender</th>
<th data-column-id="subject" data-editable="true">Subject</th>

<!-- Select editor with inline options (JSON in a data-attr) -->
<th data-column-id="status"
    data-editable="true"
    data-edit-type="select"
    data-edit-options='[{"value":0,"label":"OK"}, …]'>Status</th>
attach("#grid", {
  onCellEdit: async ({ row, column, oldValue, newValue, revert }) => {
    try {
      await savePatch(row.id, { [column.id]: newValue });
    } catch {
      revert();   // roll the cell back if the server rejects
    }
  },
});
