Tree drag-to-reparent

Set treeReparent: true and rows become draggable. Drop one row onto another to make it the new parent. Cycle attempts (dragging a parent onto its own descendant) are silently rejected. The onReparent callback fires before the move commits — return false (or a Promise resolving false) to veto. The example below logs every move into the status banner.

Drag any folder onto another to reparent it.

import { attach } from "boostgrid";

const [grid] = attach("#grid", {
  treeMode: true,
  treeReparent: true,
  onReparent: async (child, newParent, oldParent) => {
    // Optional async validation — return false to abort the move.
    const ok = await verifyMoveOnServer(child.id, newParent?.id);
    return ok;
  },
});
// Programmatic moves — same cycle/self-guard as the drag path.
grid.reparentTreeNode(42, 7);     // move 42 under 7
grid.reparentTreeNode(42, null);  // promote 42 to a root

// Inspectors
grid.getTreeDepth(42);             // 0 for roots, depth for children
grid.getTreeAncestors(42);         // [root, …, parent] in DFS order
grid.getTreeColumnId();             // id of the caret-bearing column
