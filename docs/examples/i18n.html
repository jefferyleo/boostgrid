Internationalization

options.labels holds every UI string Boostgrid renders; options.locale drives Intl-formatted numbers and dates. Row data stays as-is — i18n translates the chrome (search placeholder, page-size dropdown, pagination summary, column-toggle button, tree carets, etc.). Switch the locale below and watch the toolbar, the “Showing N” line, the date column, and the side panel update.

import { attach } from "boostgrid";

attach("#grid", {
  locale: "de-DE",
  labels: {
    infos: "Zeige {start} bis {end} von {total} Einträgen",
    search: "Suche",
    noResults: "Keine Treffer.",
    all: "Alle",
    refresh: "Aktualisieren",
    columns: "Spalten",
    searchColumns: "Spalten suchen",
    resetColumns: "Zurücksetzen",
    bulkSelected: "{n} ausgewählt",
    bulkClear: "Aufheben",
    treeExpand: "Aufklappen",
    treeCollapse: "Zuklappen",
    // …everything you don't override stays English.
  },
});
// Every translatable label, with its default value.
all:               "All"
infos:             "Showing {start} to {end} of {total} entries"
loading:           "Loading…"
