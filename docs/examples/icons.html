Icon set swap (FontAwesome)

Boostgrid ships with two ready-made icon maps — bootstrapIcons (default) and fontAwesomeIcons. Pass either to the icons option, or supply your own object of class strings.

Heads-up: the FontAwesome stylesheet is loaded on demand for this demo. In your own app, add the FA CDN <link> to your page first.
ID Sender Subject 
import { attach, fontAwesomeIcons } from "boostgrid";

attach("#grid", {
  icons: fontAwesomeIcons,
});
// Roll your own — every key falls back to bootstrapIcons if omitted
attach("#grid", {
  icons: {
    columns:    "bi-layout-three-columns",
    refresh:    "bi-arrow-clockwise",
    search:     "bi-search",
    sortAsc:    "bi-caret-up-fill",
    sortDesc:   "bi-caret-down-fill",
  },
});
import { fontAwesomeIcons } from "boostgrid";
