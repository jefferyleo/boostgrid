Multi-column sort

With multiSort: true, hold Shift and click multiple headers to combine sort orders. The grid below ships with Sender ↑ and Received ↓ applied at load.

ID Sender Subject Received 
<!-- Initial sort: data-order on each header -->
<th data-column-id="sender"   data-order="asc">Sender</th>
<th data-column-id="received" data-order="desc">Received</th>
attach("#grid", {
  multiSort: true,
});

// Programmatic sort dictionary:
grid.sort({ sender: "asc", received: "desc" });
