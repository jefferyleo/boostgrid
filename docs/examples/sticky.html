Sticky header & truncated-cell tooltips

stickyHeader: true pins the <thead> to the viewport top while you scroll the body. Pure CSS via position: sticky — coexists with frozen columns (z-indexes are arranged so the corner cell wins). Hover any clipped cell below to see the auto-generated title tooltip that truncatedTooltips adds when content overflows.

Scroll the table body — the header stays put. The --boostgrid-sticky-top CSS variable lets you offset for a fixed page nav (handy on a long-form page).

import { attach } from "boostgrid";

attach("#grid", {
  stickyHeader: true,
  truncatedTooltips: true, // default
});
/* The viewport top isn't always (0). If you have a fixed page nav,
   tell Boostgrid where the sticky header should anchor: */
.boostgrid {
  --boostgrid-sticky-top: 56px;
}

/* Truncated tooltips fire only when the cell actually clips, so
   pair with the standard CSS-truncation idiom on whichever cells
   you want clipped: */
.my-grid td {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
