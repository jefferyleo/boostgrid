External toolbar

Drive the grid from buttons that live outside it. Hold a reference to the instance returned by attach() and call its public methods — append, clear, search, reload.

ID Sender Subject Received 
<button id="add">Append row</button>
<button id="clear">Clear</button>
<button id="search">Search</button>
<button id="reload">Reload</button>

const [grid] = attach("#grid");

add.onclick    = () => grid.append([{ id: 999, sender: "new@x.io", subject: "Hot off the press" }]);
clear.onclick  = () => grid.clear();
search.onclick = () => grid.search("alpha");
reload.onclick = () => grid.reload();
