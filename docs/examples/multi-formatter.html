Combined formatters

Multiple formatters in a single grid. Status renders as a coloured pill; Amount runs through Intl.NumberFormat for currency; Received gets a human-readable relative date.

ID Sender Subject Amount Received Status 
<th data-column-id="amount"   data-formatter="money">Amount</th>
<th data-column-id="received" data-formatter="rel">Received</th>
<th data-column-id="status"   data-formatter="pill">Status</th>
const money = new Intl.NumberFormat("en-US", {
  style: "currency", currency: "USD",
});

attach("#grid", {
  formatters: {
    money(_col, row) {
      return money.format(row.amount ?? 0);
    },
    rel(_col, row) {
      const diff = (new Date(row.received) - Date.now()) / 86400000;
      return new Intl.RelativeTimeFormat("en", { numeric: "auto" })
        .format(Math.round(diff), "day");
    },
    pill(_col, row) {
      const map = { 0: ["success", "OK"], 1: ["info", "Info"],
                   2: ["warning", "Warn"], 3: ["danger", "Error"] };
      const [v, l] = map[Number(row.status)] ?? ["secondary", "?"];
      return `<span class="badge text-bg-${v}">${l}</span>`;
    },
  },
});
const formatters = { money, rel, pill };  // same fns as the vanilla tab

<ReactBoostgrid
  data={rows}
  columns={[
    { id: "amount",   text: "Amount",   formatter: formatters.money },
    { id: "received", text: "Received", formatter: formatters.rel   },
    { id: "status",   text: "Status",   formatter: formatters.pill  },
  ]}
/>
