Events log

Every grid event fires both as a typed callback (grid.on(name, fn)) and as a bubbling DOM CustomEvent on the table element (boostgrid:<name>). Try sorting, paging, searching, selecting.

ID Sender Subject

Live event stream:

const [grid] = attach("#grid", { selection: true });

["loaded", "sorted", "searched", "selected", "deselected",
 "appended", "removed", "cleared"].forEach((name) => {
  grid.on(name, (...args) => console.log(name, args));
});
// Same events also bubble as DOM CustomEvents:
grid.element.addEventListener("boostgrid:sorted", (e) => {
  console.log("sorted:", e.detail);  // detail = [sortDictionary]
});

// Or via a parent — they bubble:
document.addEventListener("boostgrid:loaded", () => {});
