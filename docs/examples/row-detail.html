Master/detail row expansion

Set rowDetail to a function returning HTML or a node, and Boostgrid adds an expand chevron in a leading affordance cell. The panel that follows the row is colspan'd across every column — perfect for a deep-link summary, a nested chart, or a related-rows mini-table.

ID Sender Subject Received 
import { attach } from "boostgrid";

const [grid] = attach("#grid", {
  rowDetail: (row) => `
    <div class="row g-3">
      <div class="col-md-6">
        <h6>${row.subject}</h6>
        <p class="text-muted small mb-0">
          From <strong>${row.sender}</strong> on ${row.received}.
        </p>
      </div>
      <div class="col-md-6">
        <dl class="row mb-0 small">
          <dt class="col-sm-4">Status</dt>
          <dd class="col-sm-8">${row.status ?? "—"}</dd>
        </dl>
      </div>
    </div>
  `,
  rowDetailExpanded: "none", // "all" | record
});
// Programmatic control
grid.toggleRowDetail(42);
grid.isRowDetailExpanded(42);  // → boolean
grid.expandAllRowDetails();
grid.collapseAllRowDetails();
