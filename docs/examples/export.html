Export · CSV / Excel / Print

The boostgrid-export companion package adds CSV, Excel, and Print export. Excel uses the optional peer dependency xlsx-js-style via a runtime dynamic import — install it only if you need .xlsx output.

ID Sender Subject Received 
$ npm install boostgrid boostgrid-export
# Optional — only needed if you call xlsx()
$ npm install xlsx-js-style
import { attach } from "boostgrid";
import { attachExport } from "boostgrid-export";

const [grid] = attach("#grid");
const exporter = attachExport(grid, {
  filename: "messages",
  include:  "filtered",    // "current" | "filtered" | "all"
});

exporter.csv();
await exporter.xlsx();
exporter.print();
<!-- attachExport also adds a delegated click listener for declarative buttons -->
<button data-bg-export="csv">Export CSV</button>
<button data-bg-export="xlsx">Export Excel</button>
<button data-bg-export="print">Print</button>
