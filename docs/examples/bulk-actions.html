Bulk action bar

When at least one row is selected, a sticky toolbar slides in above the table. bulkActions returns the user-defined contents (HTML string or node) — Boostgrid prepends the live count and a "Clear" button. Combine with multiSelect for the canonical inbox / mailbox UX.

import { attach } from "boostgrid";

const [grid] = attach("#grid", {
  selection: true,
  multiSelect: true,
  keepSelection: true,

  bulkActions: (rows) => `
    <button type="button" class="btn btn-sm btn-outline-primary"
            data-bg-action="archive">
      <i class="bi bi-archive"></i> Archive
    </button>
    <button type="button" class="btn btn-sm btn-outline-danger"
            data-bg-action="delete">
      <i class="bi bi-trash"></i> Delete
    </button>
    <span class="text-muted small ms-2">
      ${rows.reduce((sum, r) => sum + (r.unread ? 1 : 0), 0)} unread
    </span>
  `,
});

// Wire your custom data-bg-action buttons via the grid's wrapper
grid.element.parentElement?.addEventListener("click", (e) => {
  const t = e.target.closest("[data-bg-action]");
  if (t?.dataset.bgAction === "archive") {
    grid.remove(); // uses current selection
  }
});
