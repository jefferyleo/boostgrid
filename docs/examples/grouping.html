Row grouping with subtotals

Set groupBy to a column id and the body bucket-renders rows under collapsible group headers. Sorting is applied before grouping, so members within a group respect the user's sort dictionary. Add per-column groupAggregators to render subtotal footer rows.

ID Status Sender Subject Amount 
attach("#grid", {
  groupBy:       "status",
  groupExpanded: "all",    // "all" | "none" | { open: true, closed: false }
  groupAggregators: {
    amount: (_col, ctx) => {
      const sum = ctx.rows.reduce((s, r) => s + Number(r.amount || 0), 0);
      return `<strong>${sum.toFixed(2)}</strong>`;
    },
  },
});

// Toggle programmatically
grid.toggleGroup("open");
grid.isGroupExpanded("closed");
