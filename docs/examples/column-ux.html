Column UX — reorder, resize, frozen-right

Headers are draggable and resizable by default. Drag a column header left/right to rearrange, or drag the right edge of any header to resize the column. The two trailing columns (Status, Actions) are pinned with frozen: "right" — they stay anchored when the table scrolls horizontally. The new column-toggle panel (top-right) doubles as a reorder UI: drag the grip handle next to any item to reshuffle, and "Reset to defaults" reverts everything.

import { attach } from "boostgrid";

const [grid] = attach("#grid", {
  // Reorder + resize default to true; turn off per-column with
  // `reorderable: false` / `resizable: false`.
  columnReorder: true,
  columnResize: true,
  onColumnReorder: (ids) => console.log("new order", ids),
  onColumnResize: (id, px) => console.log("resized", id, px),
});

// Programmatic equivalents
grid.reorderColumn("sender", "subject", "after");
grid.resetColumnState();
