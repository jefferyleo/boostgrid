Footer / column totals

Two ways to render a <tfoot> — declaratively, with a per-column footerFormatter, or imperatively, with a single footerCallback(tfootRow, ctx) that runs after every draw. The grid below uses both: the Amount column auto-sums via footerFormatter, and a callback overrides the Subject cell with a row-count summary.

ID Sender Subject Amount 
<!-- A column with data-footer-formatter resolves through options.footerFormatters -->
<th data-column-id="amount"
    data-formatter="money"
    data-footer-formatter="sumAmount"
    data-align="right">Amount</th>

<!-- <tfoot> is auto-created when any footer signal is set; you may also
     author one explicitly to keep static cells. -->
const money = new Intl.NumberFormat("en-US", {
  style: "currency", currency: "USD",
});

const [grid] = attach("#grid", {
  formatters: {
    money: (_col, row) => money.format(Number(row.amount) || 0),
  },
  footerFormatters: {
    sumAmount: (_col, ctx) => {
      const total = ctx.filteredRows.reduce((s, r) => s + Number(r.amount || 0), 0);
      return `<strong>${money.format(total)}</strong>`;
    },
  },
  // Runs last; can read ctx and overwrite anything the formatters wrote.
  footerCallback: (tr, ctx) => {
    const subjectCell = tr.querySelector('[data-column-id="subject"]');
    if (subjectCell) {
      subjectCell.innerHTML = `<span class="text-muted small">
          Showing ${ctx.start}–${ctx.end} of ${ctx.filteredRows.length}</span>`;
    }
  },
});

// Public accessor (analog of DataTables column().footer())
grid.getFooterCell("amount");   // → <th data-column-id="amount">…
<ReactBoostgrid
  data={rows}
  columns={[
    { id: "id",      text: "ID",      identifier: true, type: "numeric" },
    { id: "sender",  text: "Sender" },
    { id: "subject", text: "Subject" },
    { id: "amount",  text: "Amount",
      formatter: (_c, r) => money.format(Number(r.amount) || 0),
      footerFormatter: (_c, ctx) =>
        money.format(ctx.filteredRows.reduce((s, r) => s + Number(r.amount || 0), 0)),
    },
  ]}
  options={{
    footerCallback: (tr, ctx) => {
      const cell = tr.querySelector('[data-column-id="subject"]');
      if (cell) cell.textContent = `Showing ${ctx.start}–${ctx.end} of ${ctx.filteredRows.length}`;
    },
  }}
/>
